導入手順

手順 1

ファイルを開いてMinecraftにインポート

ダウンロードしたファイルを開き、Minecraftにインポートします。

手順 2

アドオンをワールドに適用

ワールドの設定から、リソースパックとビヘイビアーパックの場所に行き、ワールドに適用してください。

リソースパックとビヘイビアーパック、どちらも適用しないと動きません。
手順 3

ワールド内で起動して使う

エラーや注意事項が出た場合は、コマンド欄の表示を確認してください。

注意点

  • 念のため、導入前にワールドのバックアップを取ることをおすすめします。
  • Minecraftのバージョンによっては正常に動作しない場合があります。
  • このアドオンは、対象範囲が読み込まれている前提で動作します。読み込みチャンク設定が狭い場合、正常に使えないことがあります。
